# 🏄‍♂️ Surf para Dummies

App móvil de previsión de surf para el Cantábrico, diseñada para surfistas que quieren saber si merece la pena ir a surfear sin tener que interpretar datos técnicos complejos.

## 📱 Descarga

[Descargar APK para Android](application-67e2d0d1-719e-4834-9741-831eb8be917b.apk)

## ✨ Características

- 🌊 Datos reales de olas, viento y temperatura del agua
- 📍 20 spots del Cantábrico ordenados por cercanía
- ❤️ Spots favoritos guardados en la nube
- 🔍 Buscador de spots
- 🗺️ Abrir spot en Google Maps
- 📹 Webcam del spot si disponible
- 🔐 Autenticación de usuarios

## 🛠️ Stack

- **React Native** + Expo
- **Expo Router** para navegación
- **Supabase** para base de datos y autenticación
- **NativeWind** para estilos
- **Zustand** para gestión de estado
- **Open-Meteo API** para datos meteorológicos
